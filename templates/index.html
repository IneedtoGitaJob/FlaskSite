Web Positivity

Simple website that takes in a search prompt and displays either a color depending on how positive or negative the website is (Green is positive and red is negative) or a pie chart showing the positivity and negativity of a selection of News given a number of years in a range. Positivity of a website is determined using the NLTK library. This process takes ~20 seconds.
